To clone and run this application, you'll need Git, Node.js v18.0.0 or higher + Npm v8.6.0+ (Nvm use 18.0.0) or higher installed on your computer. From your command line:

    # Clone this repository
        $ git clone https://github.com/visualpage/Tema-Limpo

    # Install dependencies
        $ npm install
        $ npm install --save-dev webpack webpack-cli webpack-stream
        $ npm install --save-dev css-minimizer-webpack-plugin sass-loader css-loader postcss-loader style-loader sass autoprefixer
        $ npm install --save-dev babel-loader @babel/core @babel/preset-env
        $ npm install --save-dev html-webpack-plugin
        $ npm install --save-dev mini-css-extract-plugin



    # Run the app
        $ gulp build
        $ gulp
